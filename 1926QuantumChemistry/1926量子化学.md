## 1926量子化学


#### 算符的概念

算符（Operator）是作用于函数或数学对象上，将其变换为另一个函数或数学对象的数学操作。可以把算符理解为"函数的函数"——它接受一个函数作为输入，输出另一个函数。在数学表示中，如果 $\hat{A}$ 是一个算符，$f(x)$ 是一个函数，那么算符作用的结果可以写成 $\hat{A}f(x)$ 或 $\hat{A}[f(x)]$。

一、常见算符类型

微分算符：最基本的微分算符是求导算符 $\frac{d}{dx}$，它将函数变换为其导数。

例子
对函数 $f(x) = x^2$ 应用微分算符：$\displaystyle \frac{d}{dx}[x^2] = 2x$
对 $g(x) = \sin(x)$ 应用：$\displaystyle \frac{d}{dx}[\sin(x)] = \cos(x)$
二阶微分算符 $\displaystyle \frac{d^2}{dx^2}$ 表示对函数求两次导数： $\displaystyle \frac{d^2}{dx^2}[x^3] = \frac{d}{dx}[3x^2] = 6x$

积分算符：积分算符 $\int$ 是微分算符的逆运算，将函数变换为其不定积分。
例子： $\displaystyle \int[2x , dx] = x^2 + C$

乘法算符：最简单的算符是乘以某个函数。例如，算符 $\hat{M}_x$ 定义为"乘以 $x$"： $\hat{M}_x[f(x)] = x \cdot f(x)$
应用到 $f(x) = \sin(x)$： $\hat{M}_x[\sin(x)] = x\sin(x)$ 

**复合算符：算符可以组合使用。例如，动量算符在量子力学中定义为 $\displaystyle \hat{p} = -i\hbar\frac{d}{dx}$（其中 $\hbar$ 是约化普朗克常数）。**


二、算符的性质

线性性：如果算符 $\hat{A}$ 满足以下两个条件，则称为线性算符：
$\hat{A}[f(x) + g(x)] = \hat{A}[f(x)] + \hat{A}[g(x)]$
$\hat{A}[cf(x)] = c\hat{A}[f(x)]$（$c$ 为常数）
例子：微分算符是线性的。 $\displaystyle \frac{d}{dx}[3x^2 + 2x] = \frac{d}{dx}[3x^2] + \frac{d}{dx}[2x] = 6x + 2$ 

非线性运算的例子是取函数的对数，例如 $\log 2x = 2\log x$ 对所有 $x$ 并不成立。
取平方的运算也是非线性的，例如 $(2x)^2 = 2x^2$ 对所有 $x$ 并不成立。

算符的交换性：两个算符 $\hat{A}$ 和 $\hat{B}$ 的对易子定义为： $[\hat{A}, \hat{B}] = \hat{A}\hat{B} - \hat{B}\hat{A}$
如果 $[\hat{A}, \hat{B}] = 0$，则称这两个算符可交换。

例子：考察位置算符 $\hat{x}$（乘以 $x$）和微分算符 $\displaystyle \frac{d}{dx}$：
对函数 $f(x)$ 先乘以 $x$ 再求导： $\displaystyle \frac{d}{dx}[xf(x)] = f(x) + x\frac{df}{dx}$
先求导再乘以 $x$： $\displaystyle x\frac{d}{dx}[f(x)] = x\frac{df}{dx}$
两者不相等，差值为 $f(x)$，因此： $\displaystyle [\frac{d}{dx}, \hat{x}]f(x) = f(x)$
这在量子力学中对应著名的正则对易关系。


三、算符在物理中的应用

在量子力学中，物理量由算符表示：
哈密顿算符（能量）：$\displaystyle \hat{H} = -\frac{\hbar^2}{2m}\frac{d^2}{dx^2} + V(x)$
动量算符：$\displaystyle \hat{p} = -i\hbar\frac{d}{dx}$
位置算符：$\displaystyle \hat{x} = x \times$（乘以 $x$）
薛定谔方程就是哈密顿算符作用于波函数的结果： $\displaystyle \hat{H}\psi(x,t) = i\hbar\frac{\partial\psi}{\partial t}$

拉普拉斯算符：在多元微积分中，拉普拉斯算符 $\nabla^2$ 或 $\Delta$ 是二阶偏微分算符： $\displaystyle \nabla^2 f = \frac{\partial^2 f}{\partial x^2} + \frac{\partial^2 f}{\partial y^2} + \frac{\partial^2 f}{\partial z^2}$
例子：对 $\displaystyle f(x,y) = x^2 + y^2$ 应用二维拉普拉斯算符： $\displaystyle \nabla^2 f = \frac{\partial^2}{\partial x^2}(x^2 + y^2) + \frac{\partial^2}{\partial y^2}(x^2 + y^2) = 2 + 2 = 4$


四、实际应用技巧

使用算符时需要注意：
作用顺序：算符从右向左依次作用。$\hat{A}\hat{B}f$ 表示先对 $f$ 应用 $\hat{B}$，再对结果应用 $\hat{A}$。 
括号的重要性：$\hat{A}(fg)$ 可能不等于 $(\hat{A}f)g$。例如微分算符： $\displaystyle \frac{d}{dx}(x \cdot \sin x) \neq \left(\frac{d}{dx}x\right) \cdot \sin x$ 
本征值问题：形如 $\hat{A}f = \lambda f$ 的方程称为本征值方程，$\lambda$ 是本征值，$f$ 是本征函数。
通过理解算符的概念，我们可以用统一的框架处理微分方程、量子力学和许多数学物理问题，使复杂的计算变得更加系统化和优雅。





#### 动量算符的由来

在量子力学（包括量子化学）中，动量算符的形式不是随意定义的，而是基于几个核心物理原理和数学要求严格推导出来的。下面我从最直观的物理图像开始，逐步解释为什么必须是这个形式。

一、德布罗意假设：粒子具有波性质
经典力学中动量是 $p = mv$。量子力学引入波粒二象性，德布罗意提出粒子关联一个波，其波长满足 $\displaystyle \lambda = \frac{h}{p}$，其中 $h$ 是普朗克常数。定义 $\hbar = h/2\pi$，波数 $k = 2\pi/\lambda$，则 $p = \hbar k$ 描述自由粒子的平面波函数通常取形式 $\displaystyle \psi(x) = A e^{i k x} = A e^{i p x / \hbar}$  这个波函数对应动量确定的粒子，本征值就是 $p$。


二、要求动量算符满足本征值方程
我们希望存在一个动量算符 $\hat{p}$，使得 $\hat{p} \psi(x) = p \psi(x)$ 
对平面波求导：$\displaystyle \frac{d}{dx} \psi(x) = i k \psi(x) = i \frac{p}{\hbar} \psi(x)$ 
两边乘以 $-i\hbar$：$\displaystyle -i\hbar \frac{d}{dx} \psi(x) = -i\hbar \cdot i \frac{p}{\hbar} \psi(x) = p \psi(x)$ 
因此，算符必须是 $\displaystyle -i\hbar \frac{d}{dx}$，才能使平面波成为动量本征函数。


三、为什么有“$-i$”？—— 厄米性（Hermitian）要求
可观测量对应的算符必须是厄米的，即期望值必须为实数，且本征值实数。
考虑算符 $\displaystyle \frac{d}{dx}$，它不是厄米的（积分时会产生边界项，导致期望值可能为虚数）。  
引入 $-i$ 后：  $\displaystyle \hat{p} = -i\hbar \frac{d}{dx}$，内积计算表明它在适当边界条件（波函数平方可积）下是厄米的，从而保证动量期望值实数，符合物理要求。如果去掉 $-i$（纯实系数），算符不再厄米，期望值会是虚数，物理上无意义。


四、为什么必须这样？—— 正则对易关系
量子力学最基本的假设是位置和动量的对易关系：$[\hat{x}, \hat{p}] = \hat{x}\hat{p} - \hat{p}\hat{x} = i\hbar$ 

在位置表象中，$\hat{x}$ 就是乘以 $x$：$\hat{x} \psi(x) = x \psi(x)$ ，假设动量算符形式为 $\displaystyle f(i\hbar \frac{d}{dx})$，代入对易关系计算，只有 $\hat{p} = -i\hbar \frac{d}{dx}$ 才能精确满足 $[\hat{x}, \hat{p}] \psi(x) = i\hbar \psi(x)$ 

简单验证： $\displaystyle \hat{p} (x\psi) = -i\hbar \frac{d}{dx}(x\psi) = -i\hbar (\psi + x \psi')$，$x \hat{p} \psi = x (-i\hbar \psi') = -i\hbar x \psi'$，相减得 $i\hbar \psi$。

如果符号或系数错了，对易关系就不成立，整个量子力学框架（包括不确定性原理）都会崩溃。

总结
动量算符 $\displaystyle \hat{p} = -i\hbar \frac{d}{dx}$ 是由以下要求共同决定的
德布罗意波关系 → 平面波本征函数推导基本形式
算符必须厄米 → 引入 $-i$ 
位置-动量正则对易关系 → 固定精确系数和符号

这是在位置表象下的表示。在动量表象中，动量算符就是乘以 $p$，位置算符变成 $\displaystyle -i\hbar \frac{d}{dp}$，形式对称。这些定义确保了量子力学与经典极限的对应（对应原理），并在量子化学计算（如分子轨道、哈密顿量）中一致应用。
